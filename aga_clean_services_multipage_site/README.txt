How to use:
1. Keep this folder inside the main project so the pages can load shared assets from `..`.
2. The pages use:
   - `../aga_clean_services_logo.svg`
   - `../Modern cleaning service logo design.png`
   - `../site.js`
3. Open `index.html` as the multipage homepage.
4. Replace every `07xxxxx` placeholder with the real phone or WhatsApp number.
5. Connect the contact form to your website form plugin, hosting provider, or email service.
6. If you publish this folder separately, copy the shared parent assets into matching relative paths or update the links.
