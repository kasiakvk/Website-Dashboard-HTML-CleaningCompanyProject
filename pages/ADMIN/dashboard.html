AGU Clean Services Admin Dashboard

This page works first as the login screen. After successful login it switches into the full admin panel for AGU Clean Services multi-page website settings, content and management.

Open this page through the local server at http://localhost:3000/ADMIN/dashboard.html. Opening the HTML file directly from disk will block admin API requests.

Login to Dashboard

Use the owner account to open website settings, lead management, review moderation and multi-page controls.

Admin Scope

What this panel controls

  • Lead inbox, quote requests and follow-up workflow.
  • Dedicated quote mailbox page connected to Gmail compose.
  • Review moderation, publication and removal.
  • Owner account and business profile settings.
  • Shared content library for services, prices, links and brand data.
  • Page-by-page settings for the AGU Clean Services multi-page website.
